Detección de transacciones bancarias fraudulentas
Proyecto final de Machine Learning para predecir diferentes tipos de datos de estrellas entre si.

Autor: David Langner

Profesora: Dra. Lorena Díaz

Institución: UAEM - IICBA - CInC (Lic. en Inteligencia Artificial)

Descripcion

En este proyecto se predeciran datos por diferentes tipos de regresion para comprara sus desempenos y a su vez,
poder ver que datos son mas utiles para predecir unos a otros.
